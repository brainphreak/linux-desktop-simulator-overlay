How It Works

The overlay mode uses CSS positioning and pointer-events to create a transparent layer that sits above your content. The taskbar and windows are interactive, but the desktop background is completely invisible and non-blocking.

You can scroll this page normally, interact with all your content, and the desktop environment only activates when you use the taskbar or open windows.
